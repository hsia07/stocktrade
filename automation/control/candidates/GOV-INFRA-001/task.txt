=== PROBLEM STATEMENT ===

Bounded auto-construction relies on the `work/candidate-*` branch naming
convention to distinguish candidate branches from other non-canonical work.
Without a machine-checkable validator, non-conformant branch names could be
pushed to the remote, creating organizational drift and bypassing governance
expectations.

=== SOLUTION ===

- Add Assert-CandidateBranchNaming function to .githooks/pre-push.ps1
- Non-canonical branch pushes must match pattern: refs/heads/work/candidate-*
- Canonical branches (main, work/canonical-*) are exempt
- Push blocked with descriptive error message if naming violated
